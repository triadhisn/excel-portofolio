# Excel Portfolio

This repository contains several Excel practice files related to data entry and basic data processing.

The datasets are created to simulate simple administrative tasks such as inputting data, calculating totals, and organizing information.

## What I did

- Input and organize data (100–200 rows)
- Use Excel formulas such as SUM, AVERAGE, IF, and VLOOKUP
- Perform basic data checking using COUNTIF and SUMIF
- Create simple summaries from data
- Use Pivot Table to group and analyze data
- Create basic charts for visualization

## Purpose

This portfolio is part of my preparation for entry-level roles such as data entry and administrative staff.

I am currently improving my Excel skills and open to learning more in a working environment.
